
	% Readme.txt for Automatica 24-0799
	% N. Yang et al, "Optimal Linear Detector Design Against False Data Injection Attacks", 2025
	% MATLAB Code

	% IMPORTANT: Please install the CVX solver before running these codes.
	% CVX can be downloaded and installed from https://cvxr.com/cvx/doc/install.html
	
	% Suggested execution order: main_system.m => main_p1.m => main_p2.m => main_p3.m => main_p4.m => main_plot.m

	Files:

	main_system.m      - Generates system matrices, Kalman filter parameters, and alarm performance parameters  
	sys.mat            - Contains data generated by main_system.m  
	kfilter.m          - Computes Kalman filter gain and covariance matrix
	main_plot.m        - plot the ROC curves and FAR/MAR points of the detectors designed by P1 to P4
	
	Folders:

	P1 (Problem I)
	 ├─ main_p1.m      - Solves Problem I
	 ├─ opt_p1.m       - Implements Theorem 2
	 └─ design1.mat    - Alarm parameters designed by main_p1.m

	P2 (Problem II)
	 ├─ main_p2.m      - Solves Problem II
	 ├─ opt_p2.m       - Implements Theorem 3
	 └─ design2.mat    - Alarm parameters designed by main_p2.m

	P3 (Problem III)
	 ├─ main_p3.m      - Solves Problem III
	 ├─ opt_p3.m       - Implements Theorem 4
	 └─ design3.mat    - Alarm parameters designed by main_p3.m

	P4 (Problem IV)
	 ├─ main_p4.m        - Solves Problem IV
	 ├─ opt_p4.m         - Implements Algorithm 1
	 ├─ opt_beta.m       - Inner loop of Algorithm 1 (called by opt_p4.m)
	 ├─ main_initialize.m- Initializes Algorithm 1
	 ├─ endpoints.mat    - Initialization data from main_initialize.m
	 ├─ erf_grad.m       - Returns gradient of the erf function
	 └─ design4.mat      - Alarm parameters designed by main_p4.m
